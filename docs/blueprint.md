## Style Guidelines:

- Primary color: Electric Indigo (#6F00FF) to give an avant-garde impression.
- Background color: Very dark gray (#121212).
- Accent color: Laser Lemon (#FFFF66) to create vibrant highlights.
- Body font: 'Inter' sans-serif, for a clean, modern, neutral look; Headline font: 'Space Grotesk', sans-serif, to give a computerized, techy feel.
- Custom, minimalist icons in neon colors to represent different sections (e.g., work experience, skills, projects).
- Dark mode design with sections divided by bright neon lines, inspired by NEON.com. Centered hero section.
- Subtle glow effects on hover for interactive elements and animated transitions between sections.
